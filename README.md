# Video Conferencing Platform

A real-time video calling web application built using WebRTC, React, Node.js, and MongoDB.

## Features
- Real-time video calling
- User authentication
- Meeting history
- Responsive UI

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- WebRTC for real-time communication
